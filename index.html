Projetos para Testes de Software

Esta aplicação web tem como objetivo auxiliar QA testers a treinar e aprender a encontrar erros, utilizando uma lista de tarefas. Também é possível utilizar a lista para a criação de uma lista de casos de teste, porém o objetivo principal é oferecer a oportunidade de conhecer suas funcionalidades de forma prática e eficiente.

Passo a Passo: Como Criar e Gerenciar Casos de Teste na Aplicação

  1. Acesse a Aplicação: Abra seu navegador e acesse o link da aplicação. Você verá uma página com uma introdução e uma lista de tarefas.
  2. Compreenda a Interface: A parte superior da página apresenta uma descrição da aplicação. Abaixo, você encontrará uma seção onde pode adicionar novos casos de teste, seguida por uma lista dos casos de teste existentes.
  3. Adicionando um Novo Caso de Teste: Localize o campo de entrada onde está escrito “Digite uma nova tarefa...”. Clique nesse campo e digite o nome do seu novo caso de teste. Depois de inserir o texto, clique no botão “Adicionar”. O novo caso de teste será adicionado à lista abaixo.
  4. Editando um Caso de Teste: Na lista de casos de teste, encontre o caso que você deseja editar. Clique duas vezes no texto do caso de teste para ativar o modo de edição. Faça as alterações necessárias e depois clique fora do campo de edição para salvar.
  5. Excluindo um Caso de Teste: Para remover um caso de teste da lista, localize o botão de exclusão (geralmente identificado como um ícone de lixeira) ao lado do caso. Clique no botão de exclusão. Uma confirmação pode aparecer; confirme a exclusão para removê-lo da lista.
  6. Filtrando Casos de Teste: Abaixo da seção de entrada, você encontrará três botões para filtrar os casos de teste: "Todas", "Concluídas" e "Pendentes". Clique em “Todas” para ver todos os casos de teste. Clique em “Concluídas” para visualizar apenas os casos que foram concluídos. Clique em “Pendentes” para ver os casos que ainda precisam ser finalizados.
  7. Revisando Seus Casos de Teste: Revise a lista de casos de teste para garantir que tudo está atualizado e organizado. Você pode usar a função de filtro para visualizar casos específicos e facilitar a revisão.
